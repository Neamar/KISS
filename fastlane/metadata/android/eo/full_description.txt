Lanĉu programojn el via hejmekrano per tajpi kelkajn leterojn.

La aplikaĵo adaptas al via uzmanieroj, permesante trovi tion, kion vi volas eĉ
pli rapide. Viaj datumoj neniam eliras vian aparaton, kaj anticipeblas konservi
baterion kaj plirapidigi vian aparaton komparata al kompleksaj kaj pezaj ĵetiloj.
KISS estas nur 250 kB kaj neniam konektiĝas al la interreto.

-------------------------
<strong>93% da uzantoj kiuj provas KISS dum semajno ankoraŭ aktive uzas ĝin post 3 jaroj.</strong>
-------------------------

Rajtoceda libera GPLv3+-programo implikas, ke ĉiuj rajtas uzi, vidi, ŝanĝi kaj dividi laŭvole, kun ĉiuj.
<a href="https://github.com/Neamar/KISS">GitHub</a>.

Helpo, tajlorado kaj ekrankopioj.
<a href="https://help.kisslauncher.com">Retejo</a>.

Tiu ĉi aplikaĵo uzas la alireblecan API de Android por ebligi la funkcion de "ŝlosi per duobla frapeto". Ĝi nek legas nek skribas iun ajn datumon.

* Uzebla privateco
* Rapida simpleco
* Praktika beleco
